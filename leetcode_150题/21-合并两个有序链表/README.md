## 🎯 核心知识点

- 链表的基本操作（遍历、创建节点、连接节点）
- 双指针技巧在链表中的应用
- 虚拟头节点（dummy node）的使用
- 边界条件处理（空链表、链表长度不同）
- 归并排序中合并两个有序序列的思想
