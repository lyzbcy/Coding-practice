## 📖 学习路径

1. **弄清题意与约束**  
   打开 `题目描述.md`，了解赎金信字符来源的限制、输入输出格式及极端数据范围。

2. **先看动画再推导公式 🎬**  
   运行 `python 动画演示.py`，观察 magazine 如何贡献字母库存、ransomNote 如何消耗库存。  
   详细操作及按键说明见 `动画演示说明.md`。

3. **系统化学习思路**  
   打开 `教学文档.md`，按章节梳理：
   - 问题分析与约束拆解
   - 错误直觉 vs 正确做法
   - 哈希计数 / 排序 / 排列比较等多种思路
   - 关键伪代码、复杂度推导与易错点

4. **对照代码实现**  
   参考 `答案.c` / `答案.cpp`，理解如何用 26 长度数组完成计数。

5. **自测与复盘**  
   将自己的代码与 `老师批作业.md` 的点评对照，最后在 `课后总结.md` 记录收获与待强化的点。

## 🎯 核心知识点

- 频次统计（计数数组 / 哈希表）在字符问题中的应用
- 只包含小写字母时，使用固定长度 26 数组的常数级优化
- 边界处理：赎金信更长、magazine 提前耗尽、字符缺失
- O(n) 线性扫描与空间 O(1) 的实现细节

## 🎬 快速开始动画演示

```bash
# 安装依赖
pip install matplotlib numpy

# 运行动画
python 动画演示.py
```

动画支持手动逐帧控制，动态展示“增加库存 → 消耗库存 → 检查是否出现负数”的过程，配合同步高亮的 C 代码行，帮助你快速理解算法逻辑。
