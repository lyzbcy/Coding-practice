# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供四个内置选项：

- 选项 1：`ransomNote="a"`, `magazine="b"`（无法构成）
- 选项 2：`ransomNote="aa"`, `magazine="ab"`（库存不足）
- 选项 3：`ransomNote="aa"`, `magazine="aab"`（可以构成）
- 选项 4：自定义输入（支持任意字符串）

## 📊 动画元素说明

- **柱状图**：显示每个字母的剩余库存
- **浅黄色柱**：当前正在处理的字母
- **浅红色柱**：库存已经变为负数（失败）
- **浅绿色条**：已确认的安全库存（在说明框中强调）
- **字符串提示**：顶部展示 `magazine` 和 `ransomNote`
- **箭头**：指示当前遍历的位置（magazine/ransom）
- **右侧代码面板**：同步高亮 C 代码行

---

## 🔁 动画流程

1. 初始化计数数组 `count[26] = 0`
2. 逐个读取 `magazine`，增加库存
3. 切换到 `ransomNote`，逐个扣减库存
4. 若出现负值，立即提示“返回 false”
5. 如果全部扣减完成且无负值，提示“返回 true”

---

## 💡 学习建议

1. **先看示例 3**，感受“库存充足 → 成功”的流程
2. **再看示例 2**，体会“扣减后出现负值”如何被捕获
3. **最后自定义输入**，练习不同字母组合及边界情况
4. 搭配 `教学文档.md` 的伪代码，一边看动画一边理清逻辑

---

## ❓ 常见问题

- **窗口无响应？**  
  请确认命令行无报错，并点击动画窗口获取焦点。

- **中文乱码？**  
  修改脚本顶部 `plt.rcParams['font.sans-serif']` 为系统可用字体。

- **柱状图太拥挤？**  
  字母种类多时可适当放大窗口或过滤输入以聚焦核心字母。

- **想导出动画？**  
  当前版本主打交互式学习，可自行添加 `plt.savefig` 保存关键步骤。
