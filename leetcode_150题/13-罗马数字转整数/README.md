   打开 `题目描述.md`，理解罗马数字的规则、输入输出与约束。

   根据需要查阅 `答案.c` 或 `答案.cpp`。

## 🎯 核心知识点

- 字符串遍历与字符映射
- 贪心思想：从左到右处理，根据下一个字符决定当前操作
- 特殊情况处理（IV、IX、XL、XC、CD、CM）
- O(n) 线性扫描的实现细节
