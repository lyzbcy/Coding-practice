## 📖 学习路径

1. **弄清题意与约束**  
   阅读 `题目描述.md`，了解高度图的含义、输入输出格式以及「左闭右闭」的雨水统计方式。

2. **先看动画再推导公式 🎬**  
   运行 `python 动画演示.py`，观察左右指针、`leftMax/rightMax` 的同步变化，弄清「短板决定水位」的原因。  
   详细操作按键见 `动画演示说明.md`。

3. **系统化学习思路**  
   打开 `教学文档.md`，按章节梳理：
   - 问题分解与错误直觉
   - 双指针、单调栈等主流解法
   - 关键状态的伪代码与推导

4. **对照代码实现**  
   参考 `答案.c` / `答案.cpp`，理解如何用常数空间维护左右指针并累计结果。

5. **自测与复盘**  
   将自己的代码与 `老师批作业.md` 的点评对照，最后在 `课后总结.md` 记录收获与待强化的点。

## 🎯 核心知识点

- 高度图化雨水统计的「木桶原理」
- 左右指针与前缀/后缀最大值的协同
- 单调栈求解柱形图类问题的通法思路
- 边界处理：首尾柱子无法蓄水、等高区间如何处理
- 时间复杂度 O(n)、空间复杂度 O(1)/O(n) 方案的取舍

## 🎬 快速开始动画演示

```bash
# 安装依赖
pip install matplotlib

# 运行动画
python 动画演示.py
```

动画支持手动逐帧控制，动态展示 `left/right` 指针、当前水位、累计雨水以及同步高亮的 C 代码行，方便对照推演。
