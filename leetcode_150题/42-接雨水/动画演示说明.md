# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供四个入口：

- 选项 1：官方示例 1  
  `height = [0,1,0,2,1,0,1,3,2,1,2,1]`

- 选项 2：官方示例 2  
  `height = [4,2,0,3,2,5]`

- 选项 3：山峰与谷地交替的数组  
  观察多个凹槽如何分批注水。

- 选项 4：自定义输入  
  支持逗号或空格分隔，至少 3 个元素。

---

## 🎮 手动控制方式

| 按键 | 功能 |
|------|------|
| 空格 / → / Enter | 下一步 |
| ← | 上一步 |
| Home | 跳到第一步 |
| End | 跳到最后一步 |
| Q / Esc | 退出程序 |

> ⚠️ 请先点击动画窗口以获取焦点，再使用键盘。

---

## 📊 动画元素说明

- **蓝色柱体**：当前高度 `height[i]`
- **青色叠层**：已经计算出的积水量
- **橙色箭头**：`left` 指针
- **绿色箭头**：`right` 指针
- **顶部信息框**：本轮行动、文字讲解与累计水量
- **右侧代码面板**：同步高亮 C 语言关键行，方便对照

---

## 🔁 动画流程

1. 初始化左右指针与 `leftMax/rightMax`
2. 比较两端高度，决定移动哪一侧
3. 在被处理的一侧叠加水量、更新总和
4. 进入下一步直至指针相遇
5. 展示“返回结果”的最终提示

---

## 💡 学习建议

1. 先用示例 1 熟悉「短板决定水位」的原理
2. 再切到示例 2，观察右侧最高值如何反复影响中间凹槽
3. 使用自定义输入测试「连续平地」「阶梯老化」等极端情况
4. 配合 `教学文档.md` 中的伪代码，将动画中的状态逐步抽象成变量

---

## ❓ 常见问题

- **窗口未响应？**  
  确认命令行中无报错，并点击图窗后再操作键盘。

- **中文乱码？**  
  根据系统字体状况修改脚本顶部的 `plt.rcParams['font.sans-serif']`。

- **想导出动图？**  
  目前版本仅支持交互式学习，如需导出可手动保存多张截图后合成 GIF。
