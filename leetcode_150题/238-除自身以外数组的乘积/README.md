## 📖 学习路径

1. **理解题意与约束**  
   先阅读 `题目描述.md`，明确不得使用除法、需要 O(n) 时间且 O(1) 额外空间的要求。
2. **配合动画建立直觉 🎬**  
   `python 动画演示.py` 手动播放两趟扫描的过程，观察前缀积、后缀积如何在一张表里累积。
   详细操作方法见 `动画演示说明.md`。
3. **深入剖析解法**  
   查看 `教学文档.md` 中的“错误直觉 vs 正确思路”“算法详解”等章节，理解如何避开 0 与除法陷阱。
4. **参考代码实现**  
   根据需要查阅 `答案.c` 与 `答案.cpp`，两份代码均实现了 O(1) 额外空间的标准写法。
5. **复盘自测**  
   完成练习后，按照 `课后总结.md` 的模板记录思路、链接与反思。

## 🎯 核心知识点

- 前缀积 / 后缀积在数组题中的通用套路
- 如何在一趟写入中复用结果数组，维持 O(1) 额外空间
- 处理 0 的出现次数对乘积的影响
- 模拟两次扫描的状态同步与边界校验
