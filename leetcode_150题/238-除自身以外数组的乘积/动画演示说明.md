# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib
```

2. **运行脚本**

```bash
python 动画演示.py
```

3. **选择示例**

程序内置四种场景：

- 选项 1：官方示例 `nums = [1,2,3,4]`
- 选项 2：单个 0 的情况 `nums = [-1,1,0,-3,3]`
- 选项 3：多个 0 `nums = [0,2,0,4]`
- 选项 4：自定义数组（空格分隔）

## 📊 动画元素说明

- **nums 行（灰色）**：原始数组，橙色描边提示当前索引。
- **answer 行（蓝/绿）**：存放前缀积与最终答案。
  - 蓝色：已写入前缀积但尚未乘后缀。
  - 绿色：前缀与后缀都已完成，即最终答案。
- **阶段提示**：左上角展示当前处于“前缀阶段/后缀阶段/结束”。
- **prefix / suffix 面板**：显示正在使用的累计乘积。
- **代码面板**：同步高亮对应的 C 语言行，方便对照。

---

## 🔁 动画流程

1. **初始化**：`answer` 置为 1，`prefix = suffix = 1`。
2. **前缀阶段**：从左到右，`answer[i] = prefix`，再更新 `prefix *= nums[i]`。
3. **后缀阶段**：从右到左，`answer[i] *= suffix`，再更新 `suffix *= nums[i]`。
4. **结束提示**：展示完整 `answer` 并提醒设置 `returnSize`。

---

## 💡 学习建议

1. 先用示例 1 熟悉两趟扫描的节奏。
2. 切换到“单个 0”场景，观察只有一个位置能得到非零答案。
3. 再试“多个 0”，理解为什么所有输出都为 0。
4. 最后输入自定义数据，手动验证前缀/后缀乘积写法。

---

## ❓ 常见问题

- **窗口无响应？**  
  关闭后重新运行，或确认当前环境支持 GUI。
- **中文乱码？**  
  将脚本开头的 `font.sans-serif` 换成本机可用字体。
- **看不懂颜色含义？**  
  图下方提供图例：蓝=前缀完成、绿=最终完成、白=未处理。
