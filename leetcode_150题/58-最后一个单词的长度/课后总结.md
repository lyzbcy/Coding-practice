思路关键点
- 使用从后往前遍历的方法，先跳过末尾空格，再计数最后一个单词
- 从字符串末尾开始，指针 `i` 初始化为 `strlen(s) - 1`
- 第一阶段：跳过末尾所有空格 `while (i >= 0 && s[i] == ' ') i--`
- 第二阶段：计数最后一个单词 `while (i >= 0 && s[i] != ' ') { length++; i--; }`
- 返回 `length` 作为最后一个单词的长度
