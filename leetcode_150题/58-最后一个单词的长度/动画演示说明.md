# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供四个内置选项：

- 选项 1：示例 1（官方示例）  
  `s = "Hello World"`

- 选项 2：示例 2（前后有空格）  
  `s = "   fly me   to   the moon  "`

- 选项 3：示例 3（多个单词）  
  `s = "luffy is still joyboy"`

- 选项 4：自定义输入（支持任意字符串）

## 📊 动画元素说明

- **浅红色区域**：已跳过的末尾空格
- **浅绿色区域**：正在计数的最后一个单词
- **浅蓝色区域**：已确认的最后一个单词（完成计数后）
- **浅灰色区域**：其他区域（前面的单词和空格）
- **红色描边**：当前指针 `i` 所在位置
- **蓝色箭头**：指针 `i`（从后往前移动）
- **␣ 符号**：表示空格字符

---

## 🔁 动画流程

1. **初始化**：提示 `i = strlen(s) - 1`，展示原始字符串
2. **跳过空格阶段**：从末尾开始，逐个跳过空格字符，指针 `i` 向前移动
3. **计数单词阶段**：找到第一个非空格字符后，开始计数，每遇到一个字母，`length++`，`i--`
4. **结束提示**：展示 `length` 的最终值（最后一个单词的长度）

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。
