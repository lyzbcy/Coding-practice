# 274. H 指数

题目来源：https://leetcode.cn/problems/h-index/

---

## 📝 题目描述

给你一个整数数组 `citations`，其中 `citations[i]` 表示研究者的第 `i` 篇论文被引用的次数。计算并返回该研究者的 **h 指数**。

根据维基百科上 h 指数的定义：h 代表"高引用次数"，一名科研人员的 h 指数是指他（她）至少发表了 `h` 篇论文，并且 **至少** 有 `h` 篇论文被引用次数大于等于 `h`。如果 `h` 有多种可能的值，h 指数是其中最大的那个。

---

## ✅ 输出规范

- 函数返回值：`h` —— 研究者的 h 指数
- h 指数满足：至少有 `h` 篇论文的引用次数 ≥ `h`，且 `h` 是满足条件的最大值

---

## 🔢 示例

### 示例 1

**输入：**  
`citations = [3,0,6,1,5]`

**输出：**  
`3`

**解释：**  
给定数组表示研究者总共有 5 篇论文，每篇论文相应的被引用了 3, 0, 6, 1, 5 次。

由于研究者有 3 篇论文每篇至少被引用了 3 次，其余两篇论文每篇被引用不多于 3 次，所以她的 h 指数是 3。

---

### 示例 2

**输入：**  
`citations = [1,3,1]`

**输出：**  
`1`

**解释：**  
有 1 篇论文的引用次数 ≥ 1，所以 h 指数是 1。

---

## 📌 提示

- `n == citations.length`
- `1 <= n <= 5000`
- `0 <= citations[i] <= 1000`

---

## 🎯 关键理解点

1. **H 指数的定义**：至少有 `h` 篇论文的引用次数 ≥ `h`
2. **最大值要求**：在所有可能的 `h` 值中，选择最大的那个
3. **边界情况**：所有论文引用次数都为 0 时，h 指数为 0
