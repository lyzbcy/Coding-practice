# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供四个内置选项：

- 选项 1：示例 1（官方示例）  
  `nums = [3,2,2,3], val = 3`

- 选项 2：示例 2（多次出现目标值）  
  `nums = [0,1,2,2,3,0,4,2], val = 2`

- 选项 3：所有元素都被移除  
  `nums = [1,1,1], val = 1`

- 选项 4：自定义输入（支持任意数组与目标值）

## 📊 动画元素说明

- **蓝色区域**：`slow` 指针已经确认的保留区（前 k 个元素）
- **灰色区域**：尚未处理或已被覆盖的区域
- **黄色描边**：当前 `fast` 指针正在比较的位置
- **卡其色描边**：本轮写入 `slow` 的位置
- **蓝色箭头**：`slow`（下一个写入位置，也是返回的 `k`）
- **绿色箭头**：`fast`（正在扫描的索引）

---

## 🔁 动画流程

1. **初始化**：提示 `slow=0`、`fast=0`，展示原始数组
2. **比较阶段**：逐个高亮 `fast` 所在元素
3. **写入阶段**：若当前值不等于 `val`，会以动画形式显示覆盖位置
4. **指针移动**：`slow` 仅在写入时前进，`fast` 每轮递增
5. **结束提示**：展示 `k=slow` 的最终值及改写后的数组

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。
