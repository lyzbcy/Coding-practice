## 📖 学习路径

1. **阅读题目要求**  
   打开 `题目描述.md`，理解题目背景、输入输出与约束。

2. **观看动画演示（强烈推荐）🎬**  
   运行 `python 动画演示.py`，直观观察栈如何匹配括号。  
   详细操作说明见 `动画演示说明.md`。

3. **深入理解思路**  
   打开 `教学文档.md`，逐步掌握：
   - 问题分析
   - 错误直觉 vs 正确思路
   - 核心算法详解（栈、匹配规则等）
   - 关键代码片段
   - 复杂度分析
   - 常见错误
   - 扩展思考

4. **查看代码实现**  
   根据需要查阅 `答案.c` 或 `答案.cpp`。

5. **复盘总结**  
   在完成题目后，填写 `课后总结.md` 模板，巩固所学。

## 🎯 核心知识点

- 栈（Stack）数据结构的使用
- 括号匹配规则与顺序
- 字符匹配与映射关系
- O(n) 线性扫描的实现细节
- 边界条件处理（空字符串、奇数长度等）

## 🎬 快速开始动画演示

```bash
# 安装依赖
pip install matplotlib numpy

# 运行动画
python 动画演示.py
```

动画支持手动逐步播放，方便你按照自己的节奏理解每一步的栈操作和字符匹配过程。
