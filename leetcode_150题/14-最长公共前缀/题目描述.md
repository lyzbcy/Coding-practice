# 14. 最长公共前缀

题目来源：https://leetcode.cn/problems/longest-common-prefix/

---

## 📝 题目描述

编写一个函数来查找字符串数组中的 **最长公共前缀**。

如果不存在公共前缀，返回空字符串 `""`。

---

## ✅ 输出规范

- 函数返回值：最长公共前缀字符串（C 语言返回 `char*`，需要动态分配内存）
- 如果所有字符串都相同，返回完整的公共前缀
- 如果没有任何公共前缀，返回空字符串 `""`

---

## 🔢 示例

### 示例 1

**输入：**  
`strs = ["flower","flow","flight"]`

**输出：**  
`"fl"`

**解释：**  
所有字符串的公共前缀是 "fl"。

---

### 示例 2

**输入：**  
`strs = ["dog","racecar","car"]`

**输出：**  
`""`

**解释：**  
输入不存在公共前缀。

---

### 示例 3

**输入：**  
`strs = ["ab", "a"]`

**输出：**  
`"a"`

**解释：**  
公共前缀是 "a"。

---

## 📌 提示

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` 仅由小写英文字母组成

---

## 💡 评测说明

评测机会按照以下流程验证你的代码：

```c
char** strs = [...];
int strsSize = ...;
char* expected = "...";  // 期望的最长公共前缀

char* result = longestCommonPrefix(strs, strsSize);
assert strcmp(result, expected) == 0;
free(result);  // 需要释放动态分配的内存
```

---

## 🔍 算法思路（伪代码）

```
方法：纵向扫描
1. 如果数组为空，返回 ""
2. 以第一个字符串为基准，遍历其每个字符位置 i
3. 对于每个位置 i：
   a. 检查第一个字符串的第 i 个字符
   b. 遍历所有其他字符串，比较它们的第 i 个字符
   c. 如果所有字符串的第 i 个字符都相同，继续
   d. 否则，返回前 i 个字符作为公共前缀
4. 如果所有字符都匹配，返回第一个字符串
```

**目标：** 找到一种 O(mn) 时间复杂度（m 为字符串平均长度，n 为字符串数量）的解法，且实现简单可靠。
