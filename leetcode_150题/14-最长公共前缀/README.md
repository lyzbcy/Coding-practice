## 📖 学习路径

1. **阅读题目要求**  
   打开 `题目描述.md`，理解题目背景、输入输出与约束。

2. **观看动画演示（强烈推荐）🎬**  
   运行 `python 动画演示.py`，直观观察如何纵向比较所有字符串的字符。  
   详细操作说明见 `动画演示说明.md`。

3. **深入理解思路**  
   打开 `教学文档.md`，逐步掌握：
   - 问题分析
   - 核心思路（纵向扫描）
   - 算法详解
   - 常见陷阱与边界处理

4. **查看代码实现**  
   根据需要查阅 `答案.c` 或 `答案.cpp`。

5. **复盘总结**  
   在完成题目后，填写 `课后总结.md` 模板，巩固所学。

## 🎯 核心知识点

- 字符串数组的遍历与比较
- 纵向扫描技巧
- 边界条件处理（空数组、空字符串、越界检查）
- 字符逐个比较的实现细节

## 🎬 快速开始动画演示

```bash
# 安装依赖
pip install matplotlib numpy

# 运行动画
python 动画演示.py
```

动画支持手动逐步播放，方便你按照自己的节奏理解每一步的字符串比较过程。
