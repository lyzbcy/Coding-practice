# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供四个内置选项：

- 选项 1：示例 1（有公共前缀）  
  `strs = ["flower","flow","flight"]`

- 选项 2：示例 2（无公共前缀）  
  `strs = ["dog","racecar","car"]`

- 选项 3：示例 3（部分匹配）  
  `strs = ["ab", "a"]`

- 选项 4：自定义输入（支持任意字符串数组，用逗号分隔）

## 📊 动画元素说明

- **浅绿色区域**：已确认的公共前缀部分
- **浅黄色区域**：当前正在比较的字符位置
- **浅红色区域**：不匹配的字符位置（公共前缀在此结束）
- **浅灰色区域**：尚未比较或不属于公共前缀的区域
- **字符串标签**：左侧显示 `strs[0]`, `strs[1]` 等
- **字符索引**：每个字符下方显示位置索引 `[0]`, `[1]` 等

---

## 🔁 动画流程

1. **初始化**：展示所有字符串，以第一个字符串为基准
2. **纵向扫描**：从位置 0 开始，逐个字符位置比较
3. **字符比较**：对于每个位置，检查所有字符串的对应字符
4. **匹配检查**：如果所有字符串的字符都匹配，继续下一个位置
5. **结束条件**：遇到不匹配或某个字符串长度不足时，返回当前公共前缀
6. **结果展示**：显示最终的最长公共前缀

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。

- **字符串显示不完整？**  
  如果字符串过长，可以调整窗口大小或修改 `figsize` 参数。
