# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供五个内置选项：

- 选项 1：示例 1（是回文数）  
  `x = 121`

- 选项 2：示例 2（负数，不是回文数）  
  `x = -121`

- 选项 3：示例 3（以 0 结尾，不是回文数）  
  `x = 10`

- 选项 4：偶数位回文数  
  `x = 1221`

- 选项 5：自定义输入（支持任意整数）

## 📊 动画元素说明

- **绿色数字框**：原始数字中已处理的前一半数字
- **灰色数字框**：原始数字中已移除的后一半数字
- **蓝色数字框**：当前 x（前一半）
- **粉色数字框**：reversed（反转的后一半）
- **代码高亮**：当前执行的代码行会以橙色背景显示

---

## 🔁 动画流程

1. **边界情况检查**：
   - 负数直接返回 `false`
   - 0 直接返回 `true`
   - 以 0 结尾且不为 0 的数返回 `false`

2. **数字反转过程**：
   - 从右到左提取每一位数字
   - 逐步构建反转数 `reversed`
   - 同时缩小原数 `x`

3. **比较判断**：
   - 偶数位数：比较 `x == reversed`
   - 奇数位数：比较 `x == reversed / 10`（去掉中间位）

4. **结果展示**：显示最终判断结果（`true` 或 `false`）

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。

- **数字太长显示不全？**  
  动画会自动调整显示范围，但如果数字位数过多，建议使用较短的测试用例。

- **为什么只反转一半？**  
  反转一半可以避免溢出问题，且只需要比较前一半和后一半即可判断回文数。
