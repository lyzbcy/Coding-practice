# 动画演示使用说明

## 📦 依赖安装

```bash
pip install matplotlib numpy
```

> 仅需轻量依赖即可运行，建议使用 Python 3.8+。

## ▶️ 运行方式

```bash
python 动画演示.py
```

启动后会在终端显示示例菜单：

1. 官方示例：`gas=[1,2,3,4,5], cost=[3,4,5,1,2]`
2. 无解示例：`gas=[2,3,4], cost=[3,4,3]`
3. 单一站点：`gas=[5], cost=[4]`
4. 自定义输入：依次输入等长的 `gas`、`cost` 数组

选择后会弹出动画窗口，并在终端回显当前数组与按键说明。

## 🎮 键盘控制

| 按键 | 作用 |
|------|------|
| 空格 / → / Enter | 下一步 |
| ← | 上一步 |
| Home | 跳转到第一步 |
| End | 跳转到最后一步 |
| Q / Esc | 关闭动画 |

> 请先 **点击动画窗口** 获取焦点，再操作键盘。

## 🎨 动画元素

- **浅蓝色方块**：当前假设的起点 `start`
- **紫色方块**：普通站点
- **橙色描边 + 箭头**：当前正在访问的站点索引
- **信息面板**：实时展示 `total`、`tank`、`start` 数值
- **粉色提示框**：当 `tank < 0` 时提示“下一轮重置起点并清空油箱”
- **右侧代码面板**：同步高亮 C 语言实现的对应行

## 🔁 流程说明

1. **初始化**：解释 `total / tank / start` 的含义
2. **遍历每个站点**：显示 `gain = gas[i] - cost[i]` 及其对油量的影响
3. **油箱为负**：提示重置起点、油箱清零，方便理解贪心切割
4. **完成一圈**：根据 `total` 判断是否存在解，并显示要返回的起点

## 💡 学习建议

1. **先看示例 1**：熟悉贪心的完整过程
2. **再看示例 2**：感受 `total < 0` 无解的情形
3. **尝试自定义**：比如让某段“巨额亏油”验证起点重置的必要性
4. **配合教学文档**：对照伪代码观察 `tank`、`start` 的更新细节

## ❓ 常见问题

- **窗口没有响应键盘？**  
  点击一次动画窗口即可。若仍无效，可在终端重新运行脚本。

- **中文字体报错或显示方块？**  
  修改脚本顶部的 `plt.rcParams['font.sans-serif']` 为本机可用字体。

- **想导出每一步截图？**  
  可在 `_draw_step` 尾部添加 `plt.savefig(...)` 保存关键帧。
