## 📖 学习路径

1. **读题 + 抓约束**  
   打开 `题目描述.md`，确认汽油与消耗数组的含义、输出规范与“唯一解”条件。
2. **先看动画再推公式 🎬**  
   运行 `python 动画演示.py`，通过油量柱状图理解 “油箱为负就重置起点” 的贪心判断。  
   详细键位与自定义输入说明见 `动画演示说明.md`。
3. **系统拆解思路**  
   `教学文档.md` 会按照 “问题分析 → 错误直觉 → 正确思路 → 代码实现 → 常见错误” 的顺序梳理。
4. **敲定代码实现**  
   根据需要查阅 `答案.c` / `答案.cpp`，两份代码与教学文档、动画中的关键变量保持一致。
5. **复盘与再练**  
   完成题目后，记得在 `课后总结.md` 中记录题解链接与反思要点，再对照「相关题目」加练。

## 🎯 核心知识点

- 前缀和 + 贪心判定：`total` 判断是否存在解，`tank` 判断当前起点是否可行
- 环路问题的起点重置技巧：油箱为负时将起点设为下一站并清空油箱
- 线性扫描下的状态维护与数组遍历细节
- 时间复杂度 O(n)、空间复杂度 O(1) 的实现套路

## 🎬 快速开始动画演示

```bash
# 安装依赖
pip install matplotlib numpy

# 运行动画
python 动画演示.py
```

动画支持键盘逐帧控制，信息面板同步显示 `total`、`tank` 与 `start` 的最新值，方便对照源码理解每一次重置的原因。
