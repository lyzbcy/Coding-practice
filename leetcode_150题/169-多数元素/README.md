## 🎯 核心知识点

- Boyer-Moore 投票算法：O(n) 时间、O(1) 空间的经典解法
- 多数元素的数学性质：出现次数超过一半的元素必然存在
- 候选元素与计数器的维护策略
- 多种解法的复杂度权衡（哈希表、排序、投票算法）
