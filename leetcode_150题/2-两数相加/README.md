## 📖 学习路径

1. **阅读题目要求**  
   打开 `题目描述.md`，理解题目背景、输入输出与约束。

2. **观看动画演示（强烈推荐）🎬**  
   运行 `python 动画演示.py`，直观观察两个链表如何逐位相加并处理进位。

3. **深入理解思路**  
   打开 `教学文档.md`，逐步掌握：
   - 问题分析
   - 核心思路（模拟竖式加法）
   - 算法详解
   - 常见陷阱与边界处理

4. **查看代码实现**  
   根据需要查阅 `答案.c` 或 `答案.cpp`。

5. **复盘总结**  
   在完成题目后，填写 `课后总结.md` 模板，巩固所学。

## 🎯 核心知识点

- 链表的基本操作（遍历、创建节点）
- 模拟竖式加法的实现
- 进位处理技巧
- 虚拟头节点（dummy node）的使用
- 边界条件处理（链表长度不同、最后有进位）

## 🎬 快速开始动画演示

```bash
# 安装依赖
pip install matplotlib numpy

# 运行动画
python 动画演示.py
```

动画支持手动逐步播放，方便你按照自己的节奏理解每一步的链表变化和进位过程。
