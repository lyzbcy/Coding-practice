## 🎯 核心知识点

- 非递减数组上的双指针去重技巧
- 原地覆盖写入与 O(1) 额外空间
- 循环不变量：`nums[0..slow]` 始终存放唯一元素
- 典型边界：空数组、全相同、全不同
