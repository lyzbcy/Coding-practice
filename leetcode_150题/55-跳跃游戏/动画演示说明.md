# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供五个内置选项：

- 选项 1：示例 1（可以到达）  
  `nums = [2,3,1,1,4]`

- 选项 2：示例 2（无法到达）  
  `nums = [3,2,1,0,4]`

- 选项 3：示例 3（已在终点）  
  `nums = [0]`

- 选项 4：示例 4（无法到达）  
  `nums = [1,0,1,0]`

- 选项 5：自定义输入（支持任意数组）

## 📊 动画元素说明

- **蓝色区域**：可达区域（`maxReach` 范围内的位置）
- **绿色区域**：当前位置 `i`
- **灰色区域**：不可达区域（超过 `maxReach` 的位置）
- **橙色箭头**：从当前位置能到达的范围
- **蓝色箭头**：`maxReach`（当前能到达的最远位置）
- **绿色箭头**：当前位置 `i`
- **红色描边**：当前正在处理的位置

---

## 🔁 动画流程

1. **初始化**：提示 `maxReach=0`，展示原始数组
2. **遍历阶段**：逐个高亮当前位置 `i`
3. **可达性检查**：如果 `i > maxReach`，说明无法到达，返回 `false`
4. **更新阶段**：更新 `maxReach = max(maxReach, i + nums[i])`
5. **提前终止**：如果 `maxReach >= numsSize - 1`，返回 `true`
6. **结束提示**：展示最终结果及 `maxReach` 的值

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。
