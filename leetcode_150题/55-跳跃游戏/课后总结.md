思路关键点
- 使用贪心算法，维护当前能到达的最远位置 maxReach
- 遍历数组，对于每个位置 i：
  - 如果 i > maxReach，说明无法到达位置 i，返回 false
  - 否则，更新 maxReach = max(maxReach, i + nums[i])
  - 如果 maxReach >= numsSize - 1，说明可以到达终点，返回 true
