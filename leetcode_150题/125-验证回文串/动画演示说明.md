# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供五个内置选项：

- 选项 1：示例 1（官方示例，是回文串）  
  `s = "A man, a plan, a canal: Panama"`

- 选项 2：示例 2（不是回文串）  
  `s = "race a car"`

- 选项 3：示例 3（空字符串）  
  `s = " "`

- 选项 4：简单示例  
  `s = "race a car"`

- 选项 5：自定义输入（支持任意字符串）

## 📊 动画元素说明

- **绿色区域**：`left` 指针当前位置（字母数字字符）
- **蓝色区域**：`right` 指针当前位置（字母数字字符）
- **红色描边**：`left` 指针正在比较的位置
- **蓝色描边**：`right` 指针正在比较的位置
- **浅红色区域**：当前跳过的非字母数字字符位置
- **灰色区域**：已处理完成的区域
- **米色区域**：待处理的字母数字字符
- **浅黄色区域**：待处理的非字母数字字符
- **绿色箭头**：`left` 指针
- **蓝色箭头**：`right` 指针

---

## 🔁 动画流程

1. **初始化**：提示 `left=0`、`right=len(s)-1`，展示原始字符串

2. **跳过非字母数字字符**：
   - 如果 `s[left]` 不是字母数字，`left` 向右移动
   - 如果 `s[right]` 不是字母数字，`right` 向左移动

3. **比较字符**：
   - 将 `s[left]` 和 `s[right]` 转为小写后比较
   - 显示比较结果（匹配或不匹配）

4. **指针移动**：
   - 如果匹配，`left++`，`right--`，继续下一轮
   - 如果不匹配，显示结果并结束

5. **结束提示**：展示最终判断结果（`true` 或 `false`）

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。

- **字符串太长显示不全？**  
  动画会自动调整显示范围，但如果字符串过长，建议使用较短的测试用例。
