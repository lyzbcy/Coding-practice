## 🎯 核心知识点

- **双指针技术**（三指针）
- **从后往前合并**的技巧
- **原地算法**（O(1) 空间复杂度）
- **时间复杂度优化**（O(m+n)）

## 💡 解题要点

1. 关键洞察：从后往前合并，避免覆盖未处理的元素
2. 三指针：`i`（nums1有效元素）、`j`（nums2）、`k`（合并位置）
3. 边界处理：确保 nums2 的所有元素都被处理
