# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供五个内置选项：

- 选项 1：示例 1（最小跳跃次数: 2）  
  `nums = [2,3,1,1,4]`

- 选项 2：示例 2（最小跳跃次数: 2）  
  `nums = [2,3,0,1,4]`

- 选项 3：示例 3（最小跳跃次数: 3）  
  `nums = [1,1,1,1]`

- 选项 4：示例 4（最小跳跃次数: 0）  
  `nums = [1]`

- 选项 5：自定义输入（支持任意数组）

## 📊 动画元素说明

- **蓝色区域**：边界内（上一次跳跃能到达的区域）
- **黄色区域**：可达但需跳跃（超过边界但可以通过跳跃到达）
- **绿色区域**：当前位置 `i`
- **灰色区域**：不可达区域（超过 `maxReach` 的位置）
- **橙色箭头**：从当前位置能到达的范围
- **蓝色箭头**：`end`（上一次跳跃的边界）
- **紫色箭头**：`maxReach`（当前能到达的最远位置）
- **绿色箭头**：当前位置 `i`
- **红色描边**：当前正在处理的位置

---

## 🔁 动画流程

1. **初始化**：提示 `jumps=0, end=0, maxReach=0`，展示原始数组
2. **遍历阶段**：逐个高亮当前位置 `i`
3. **更新阶段**：更新 `maxReach = max(maxReach, i + nums[i])`
4. **边界检查**：如果 `i == end`，说明到达边界，需要再跳一次
   - 更新 `jumps++`
   - 更新 `end = maxReach`
5. **结束提示**：展示最终结果及最小跳跃次数

## 💡 学习建议

1. **第一次观看**：选择示例 1，观察 `end` 和 `maxReach` 如何逐步更新，以及何时增加跳跃次数
2. **第二次**：尝试示例 2，理解边界跳跃法的核心思想
3. **第三次**：输入自定义测试，例如 `[1,2,3,4,5]` 或 `[3,2,1,0,4]`
4. **最后**：配合 `教学文档.md` 手动模拟代码流程，理解为什么边界跳跃法是正确的

---

## 🎯 关键理解点

- **边界 `end`**：上一次跳跃能到达的最远位置
- **`maxReach`**：在当前边界内，能到达的最远位置
- **跳跃时机**：当 `i == end` 时，说明已经到达上一次跳跃的边界，必须再跳一次才能继续前进
- **贪心策略**：在边界内，总是选择能跳得最远的位置（通过更新 `maxReach` 实现）

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。
