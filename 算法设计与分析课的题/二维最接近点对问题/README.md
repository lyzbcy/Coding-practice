## 📖 学习路径

1. **阅读题目要求**  
   打开 `题目描述.md`，理解题目背景、输入输出与约束。

2. **观看动画演示（强烈推荐）🎬**  
   运行 `python 动画演示.py`，直观观察分治算法如何递归求解最近点对。  
   详细操作说明见 `动画演示说明.md`。

3. **深入理解思路**  
   打开 `教学文档.md`，逐步掌握：
   - 问题分析
   - 分治算法核心思路
   - 算法详解与递归过程
   - 合并阶段的处理技巧
   - 常见陷阱与边界处理

4. **查看代码实现**  
   根据需要查阅 `答案.c` 或 `答案.cpp`。

5. **复盘总结**  
   在完成题目后，填写 `课后总结.md` 模板，巩固所学。

## 🎯 核心知识点

- 分治算法（Divide and Conquer）的应用
- 递归思想与递归终止条件
- 平面几何中的距离计算
- 排序在分治中的关键作用
- 合并阶段的优化技巧（条带区域）
- 时间复杂度分析：O(n log² n) 与 O(n log n)

## 🎬 快速开始动画演示

```bash

# 运行动画

```

动画支持手动逐步播放，方便你按照自己的节奏理解分治算法的每一步递归过程。
