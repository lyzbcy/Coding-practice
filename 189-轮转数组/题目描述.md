# 189. 轮转数组

题目来源：https://leetcode.cn/problems/rotate-array/

---

## 📝 题目描述

给你一个数组，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

**示例 1：**

```
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]
```

**示例 2：**

```
输入：nums = [-1,-100,3,99], k = 2
输出：[3,99,-1,-100]
解释:
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]
```

---

## ✅ 输出规范

- 函数无返回值，需要**原地修改**数组 `nums`。
- 数组中的元素向右轮转 `k` 个位置。
- 必须使用 O(1) 的额外空间（进阶要求）。

---

## 🔢 示例

### 示例 1

**输入：**  
`nums = [1,2,3,4,5,6,7]`, `k = 3`

**输出：**  
`[5,6,7,1,2,3,4]`

**解释：**  
向右轮转 3 个位置，后 3 个元素 `[5,6,7]` 移动到前面，前 4 个元素 `[1,2,3,4]` 移动到后面。

---

### 示例 2

**输入：**  
`nums = [-1,-100,3,99]`, `k = 2`

**输出：**  
`[3,99,-1,-100]`

**解释：**  
向右轮转 2 个位置，后 2 个元素 `[3,99]` 移动到前面，前 2 个元素 `[-1,-100]` 移动到后面。

---

### 示例 3

**输入：**  
`nums = [1,2]`, `k = 3`

**输出：**  
`[2,1]`

**解释：**  
数组长度为 2，轮转 3 次等价于轮转 `3 % 2 = 1` 次。

---

## 📌 提示

- `1 <= nums.length <= 10^5`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `0 <= k <= 10^5`

---

## 💡 进阶要求

- 尽可能想出更多的解决方案，至少有**三种**不同的方法可以解决这个问题。
- 你可以使用空间复杂度为 **O(1)** 的**原地**算法解决这个问题吗？
