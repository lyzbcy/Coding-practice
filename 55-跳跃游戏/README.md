## 📖 学习路径

1. **阅读题目要求**  
   打开 `题目描述.md`，理解题目背景、输入输出与约束。

2. **观看动画演示（强烈推荐）🎬**  
   运行 `python 动画演示.py`，直观观察贪心算法如何维护最大可达位置。  
   详细操作说明见 `动画演示说明.md`。

3. **深入理解思路**  
   打开 `教学文档.md`，逐步掌握：
   - 问题分析
   - 贪心算法核心思路
   - 算法详解与示例
   - 常见错误与边界处理

4. **查看代码实现**  
   根据需要查阅 `答案.c` 或 `答案.cpp`。

5. **学习错误案例**  
   运行 `python 错误代码演示.py`，了解常见错误和调试方法。

6. **复盘总结**  
   在完成题目后，填写 `课后总结.md` 模板，巩固所学。

## 🎯 核心知识点

- 贪心算法的应用
- 最大可达位置的维护
- 线性扫描与提前终止
- O(n) 时间复杂度优化
