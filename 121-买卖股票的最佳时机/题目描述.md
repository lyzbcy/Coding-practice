# 121. 买卖股票的最佳时机

题目来源：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/

---

## 📝 题目描述

给定一个数组 `prices`，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0`。

---

## ✅ 输出规范

- 函数返回一个整数，表示能获得的最大利润。
- 如果无法获得利润（即所有价格都是递减的），返回 `0`。
- 必须在买入之后才能卖出（不能在同一天买入和卖出）。

---

## 🔢 示例

### 示例 1

**输入：**  
`prices = [7,1,5,3,6,4]`

**输出：**  
`5`

**解释：**  
在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5。  
注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

---

### 示例 2

**输入：**  
`prices = [7,6,4,3,1]`

**输出：**  
`0`

**解释：**  
在这种情况下, 交易无法完成, 所以最大利润为 0。

---

### 示例 3

**输入：**  
`prices = [1,2,3,4,5]`

**输出：**  
`4`

**解释：**  
在第 1 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 5）的时候卖出，最大利润 = 5-1 = 4。

---

### 示例 4

**输入：**  
`prices = [2,4,1]`

**输出：**  
`2`

**解释：**  
在第 1 天（股票价格 = 2）的时候买入，在第 2 天（股票价格 = 4）的时候卖出，最大利润 = 4-2 = 2。

---

## 📌 提示

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

---

```c
int prices[] = {7, 1, 5, 3, 6, 4};
int expected = 5;

int result = maxProfit(prices, 6);

```

---

## 💡 核心要求

- 只能进行一次交易（买入一次，卖出一次）
- 必须在买入之后才能卖出
- 目标是找到买入和卖出的最佳时机，使利润最大化

---

**目标：** 找到一种 O(n) 时间复杂度、O(1) 额外空间的解法。
