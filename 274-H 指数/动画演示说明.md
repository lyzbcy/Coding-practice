# 动画演示使用说明

## 🎬 如何运行动画

1. **安装依赖**

```bash
pip install matplotlib numpy
```

2. **运行程序**

```bash
python 动画演示.py
```

3. **选择示例**

程序提供五个内置选项：

- 选项 1：示例 1（官方示例）  
  `citations = [3,0,6,1,5]`，答案：3

- 选项 2：示例 2（官方示例）  
  `citations = [1,3,1]`，答案：1

- 选项 3：所有论文都满足  
  `citations = [5,4,3,2,1]`，答案：3

- 选项 4：所有引用为 0  
  `citations = [0,0,0]`，答案：0

- 选项 5：自定义输入（支持任意数组）

## 📊 动画元素说明

- **绿色区域**：已确认满足 H 指数条件的论文（引用次数 ≥ 对应的 h 值）
- **黄色区域**：当前正在检查的论文
- **灰色区域**：尚未检查的论文
- **红色描边**：当前检查的位置 `i`
- **蓝色文字**：每个位置对应的 h 候选值（i+1）

---

## 🔁 动画流程

1. **初始状态**：展示原始数组
2. **排序阶段**：显示降序排序后的数组
3. **遍历检查**：逐个检查每个位置，判断 `citations[i]` 是否 ≥ `i+1`
4. **找到 H 指数**：当 `citations[i] < i+1` 时，确定 H 指数为 `i`
5. **结束提示**：展示最终的 H 指数值

## ❓ 常见问题

- **键盘无响应？**  
  请确保点击动画窗口使其获得焦点。如果仍无效，尝试重新运行或切换 Python 终端。

- **中文显示异常？**  
  修改脚本中的 `plt.rcParams['font.sans-serif']`，换成系统可用字体。

- **无法显示窗口？**  
  在无图形界面的环境中，可以使用 `plt.savefig` 导出每一步，但更推荐在本地运行。
